# Погодное приложение

Простое веб-приложение для отображения погоды в разных городах.

## Основные функции

- Добавление виджетов погоды по названию города
- Добавление виджетов по координатам (широта и долгота)
- Отображение температуры, описания погоды, времени в локации, ветра и влажности
- Карта местоположения для каждого виджета
- Автоматическое сохранение добавленных городов

## Как использовать

1. Введите название города в поле и нажмите "Добавить по городу"
2. Или введите широту и долготу и нажмите "Добавить по координатам"
3. Нажмите "Показать карту" для просмотра местоположения на карте
4. Нажмите на крестик (×) для удаления виджета

## Особенности

- Адаптивный дизайн для мобильных устройств
- Фон виджетов меняется в зависимости от погоды
- Все данные сохраняются в браузере между посещениями

Приложение работает с открытым API OpenWeatherMap.
